# Facebook-Clone

This is a C++ project that simulates a simple social media platform. It shows how user profiles, posts, comments, friends, page timelines, and memory posts can be managed using object-oriented programming. It's a hands-on example to practice classes, inheritance, and dynamic memory in C++.

Features:

Create and manage user profiles, friends, and liked pages.
Post, like, and comment on posts for users and pages.
View timelines, home feed, and memory posts.
Practice inheritance, polymorphism, and memory management.
Load data from files for users, posts, pages, comments, and activities.

Purpose:

To learn and practice object-oriented programming in C++ by simulating a small social media platform.
